## PURPOSE
Load testing services for comparison to see which language/framework can higher traffic

# Notes
- `load-test/`
    - contains locust test that will hit /users endpoint

- `python-compares/` ( mimic golang webservice app for comparison)
    - Flask app (w/Gunicorn WSGI)
    - Quart (w/Hypercorn ASGI)
    - Tornado (no gateway)
    

- `simple-golang-examples/`
    - examples on using golang, strings, collections, loops, etc.

- `webservice/`
    - simple golang webservice that creates and retrieves users


# Results:
100 users are spawned when webservice starts. All service will be using a standard for loop to get the user to return to the UI.
- Locust attributes:
    - hit `/users/{id}`
    - 1000 users
    - 1000 spawn/second
    - localhost:5000 (hit running docker)
